## Mini Sistema fictício de pagamento

## Instruções para rodar o projeto

Use o comando <kbd>docker-compose up</kbd> e após o docker subir seu container, voce precisa rodar os comandos <kbd>docker-compose exec app php artisan migrate</kbd>
e depois <kbd>docker-compose exec app php artisan db:seed</kbd>

## Outras informações

A aplicação está usando a porta 80
